# Lab 04 - Maximum Likelihood Estimation

*(Note: Although the directory is named `lab04_variational_autoencoders`, the contents of this lab focus on Maximum Likelihood Estimation (MLE) for Bernoulli and Normal probability distributions.)*

This lab focuses on exploring probability density functions and the concept of Maximum Likelihood Estimation (MLE). Students will visualize probabilistic outcomes and watch how MLE approximations converge on actual population parameters with increasing data sizes.

## Lab Structure
- `data.py`: Code for synthesizing data reflecting explicit probabilistic distributions (Bernoulli and Normal data generations).
- `visual.py`: Contains functions to plot log-likelihood curves, model convergence mappings, and data/distribution comparisons.
- `task0.py`: Script exploring the raw Bernoulli probability (coin tosses) and creating a log-likelihood mapping across range estimates.
- `task1.py`: Demonstrates ML parameter convergence against an actual "true probabilty" (p=0.7) as the simulated sample size is increased.
- `task2.py`: Fits synthetic Normal distribution data and displays the corresponding PDF distribution based on our manually calculated sample metrics vs maximum likelihood metrics.

## Setup and Execution
Execute scripts using the specified class virtual environment:

```sh
source ~/venv/sem-2/bin/activate

# Task 0: Bernoulli Distribution Coin Tosses and Log Likelihood Map
python labs/lab04_variational_autoencoders/task0.py

# Task 1: Demonstrate MLE Convergence with Increased Data
python labs/lab04_variational_autoencoders/task1.py

# Task 2: View normal distribution curves over our simulated data (calculating unbiased vs biased standard dev bounds)
python labs/lab04_variational_autoencoders/task2.py
```

Note: Code sections utilizing `matplotlib` generation will suspend terminal output as interactive modal GUI windows appear. The scripts will stall at `plt.show()` so manually close the picture graph generated by the window manager before continuing.
