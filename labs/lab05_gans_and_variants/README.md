# Lab 05 - Gaussian Mixture Models & Expectation Maximization

*(Note: Although the directory is named `lab05_gans_and_variants`, the contents of this lab focus on normal probability distributions and Gaussian Mixture Models extracted via the Expectation-Maximization algorithm.)*

This lab focuses on exploring single and multivariate Normal distributions alongside probabilistic clustering tools like Gaussian Mixture Models (GMM) using the Expectation-Maximization (EM) algorithm.

## Lab Structure
- `data.py`: Code for synthesizing electrical distribution arrays and creating grouped clusters via Multivariate distributions.
- `visual.py`: Contains functions to plot normal density gradients, generate surface contours, and map predicted spatial clusters.
- `task0.py`: Fits explicit electricity usage metrics to a standard 1D Normal distribution probability mapping.
- `task1.py`: Assesses standard variance covariances against distinct features mapping a 2D Multivariate Gaussian contour graph.
- `task2.py`: Generates two independent Normal distribution clusters and estimates cluster predictions using the EM algorithm.

## Setup and Execution
Execute scripts using the specified class virtual environment:

```sh
source ~/venv/sem-2/bin/activate

# Task 0: 1D Electricity Usage Normal Distribution Fitting
python labs/lab05_gans_and_variants/task0.py

# Task 1: 2D Multivariate Contour Graph
python labs/lab05_gans_and_variants/task1.py

# Task 2: Expectation Maximization clustering (GMM)
python labs/lab05_gans_and_variants/task2.py
```

Note: Code sections utilizing `matplotlib` generation will suspend terminal output as interactive modal GUI windows appear. The scripts will stall at `plt.show()` so manually close the picture graph generated by the window manager before continuing.
