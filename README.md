# Advanced Machine Learning – Lab Work

This repository contains implementations and experiments performed as part
of the **Advanced Machine Learning (AML)** course.

The work includes kernel methods, model training, evaluation, and visualization,
implemented using Python and standard ML libraries.

---

## Repository Structure
- labs/ : Individual lab exercises
- data/ : Datasets (raw and processed)
- models/ : Saved models
- logs/ : Error logs and debugging notes
- src/ → Common entry points and utilities
